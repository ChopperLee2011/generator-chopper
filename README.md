# Chopper Generator

`generator-chopper` is a [Yeoman](http://yeoman.io) generator that scaffolds out
a npm module application.

## Features
 - Babel
 - Standardjs
 - Travis CI
 - Mocha
 - Github-changes
 
## Getting Started

 * Install: `npm install -g generator-chopper`
 * Run `yo chopper`
